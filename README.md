# funk-tional-lean

This repository was developed to provide a short workshop style introduction to Lean. The content was originally developed for IWSLA 2025, hosted by BITS Goa. 

Installation instructions: 
1. Go to VS Code.
2. Choose the clone a github repository option.
3. Once it is cloned locally, open any .lean inside the FunctionProgrammingInLean folder and check if it is working.
4. If installation still remains an issue, you can copy-paste the code to https://live.lean-lang.org/
